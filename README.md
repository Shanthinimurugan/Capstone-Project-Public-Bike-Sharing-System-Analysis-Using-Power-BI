# 🚲 Public Bike Sharing System Analysis using Power BI

A Public Bike Sharing System enables users to rent and return bikes at designated stations across a city. This project analyzes bike station operations, availability, and utilization to uncover insights that support efficient system management.

## 📌 Project Overview

This project uses **Power BI** to analyze operational data from a public bike-sharing system. The dashboard evaluates station performance, bike and dock availability, utilization rates, station status, banking facilities, and bonus stations. Through interactive visualizations, the project helps identify usage patterns, operational bottlenecks, and opportunities to improve the efficiency of the bike-sharing network.
---

# 🎯 Project Objectives

- Analyze bike and bike stand availability across stations.
- Monitor station capacity and utilization.
- Identify frequently full and empty stations.
- Evaluate station operational status (Open/Closed).
- Compare station performance across different cities (contracts).
- Analyze banking-enabled and bonus stations.
- Identify hourly availability trends.
- Build an interactive Power BI dashboard to support operational decision-making. :contentReference[oaicite:1]{index=1}

# 🚩 Problem Statement


Public bike-sharing operators face challenges in maintaining a balanced distribution of bikes across stations due to varying demand, limited station capacity, and operational constraints. These issues can lead to stations running out of bikes or docking spaces, reducing service quality and customer satisfaction. This project addresses these challenges by analyzing operational data to identify performance gaps and support data-driven decisions for effective bike redistribution and station management.

# 📈 Key Insights

The analysis revealed that the bike-sharing network consists of 2,856 station records across 1,028 unique stations, with 92.79% of stations operational. Lyon has the largest bike-sharing network, while Marseille experiences the highest number of unavailable bikes and bike stands, indicating a need for improved bike redistribution. High-capacity cities such as Lyon, Toulouse, and Bruxelles-Capitale are expected to continue experiencing high demand, whereas stations with high bike utilization may require proactive redistribution during peak periods. These insights highlight the importance of optimizing bike redistribution, expanding infrastructure in high-demand locations, improving maintenance, and leveraging hourly availability trends to enhance the overall efficiency and reliability of the public bike-sharing system.
